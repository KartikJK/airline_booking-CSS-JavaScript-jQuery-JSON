# airline_booking-CSS-JavaScript-jQuery-JSON

Web based application that allows users to book their chosen seats on an aircraft
Using CSS, JavaScript and jQuery
Results are modelled on EasyJet system
This project is aimed to allow users to book seats, undo selection, when all seats selected proceed, get visual feedback, preventing selection of unavailable seats, see a count and list of seats booked including row and seat number, to view the seat price zones on the fixed layout or as the mouse hovers over seats.
